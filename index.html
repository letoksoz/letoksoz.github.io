NeuroD2 Project:

The aim of this project is to identify and characterize NeuroD2 protein binding sites on DNA using machine learning algorithms, such as logistic regression, neural networks and convolutional neural networks, which will enable us to better understand the functional role of NeuroD2 protein in neuronal differentiation. These algorithms are implemented using the Knet package in Julia. Data is obtained by combining several different datasets, including ChIP-Seq Data and Histone Binding Data. Different bioinformatic techniques are used to model the position of a NeuroD2 protein within DNA. Also, we implement window architecture, which uses a similar logic to RNN, to our machine learning models to represent the relationship between NeuroD2 protein and its interacting partners. Accounting Paper Project:

Title: "Machine Learning and Financial Distress Prediction"

The aim of this project is to identify factors correlating with financial distress for the firms listed on the Borsa Istanbul Stock Exchange using machine learning algorithms, such as logistic regression, neural networks, beehive algorithm and random forests, which will enable us to develop a model to investigate the effect of certain financial ratios on the likelihood of financial distress. We have been implementing machine learning algorithms using Python. The repository is not yet available for public use. The short-term goal is to submit this paper to conferences in order to get feedback on the findings before ultimately submitting it to a journal.

Honors Thesis:

Title: "21 cm Hydrogen Line and Its Power Spectrum"

Advisor: Professor Dragan Huterer (University of Michigan - Ann Arbor, Department of Physics)

In this thesis, I examined the 21 cm Hydrogen Line and utilized statistical techniques, such as the Fisher Matrix, to forecast the precision of cosmological parameters inherent to the model in the presence of noise, using Python and CAMB sources.
