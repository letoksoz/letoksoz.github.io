My Current Projects

NeuroD2 Project:

The aim of this project is to identify and characterize binding sites of NeuroD2 protein on DNA using machine learning algorithms such as Logistic Regression, Neural Networks and Convolutional Neural Networks, which will enable us to understand the functional role of NeuroD2 protein in neuronal differentiation. The data was obtained by combining several different data sets such as ChIP-Seq Data and Histone Binding Data. Preprocessing the data was one of crucial and challenging parts of this project,since the data was too large and sparse. Special mini batching technique called oversampling is employed to obtain a balanced data sets(about equal number of positve and negative samples). The code can be found here.

I have been using Julia and Knet package to implement machine learning algorithms. Here is a sample log regression algorithm I wrote in Julia using Knet package.(Ml.jl file)

Accounting Paper Project:

Title: "Machine Learning and Financial Distress Prediction"

Our aim is to identify factors correlating with financial distress of the firms listed on the Borsa Istanbul Stock exchange using Machine learning Algorithms such as Logistic Regression,Neural Networks,Beehive algorithm and random forests,which will enable us to develop a model to investigate the effect of financial ratios on the likelihood of financial distress. We have been implementing machine learning algorithms using Python. The repository is not available for public use. Our short term goal is to submit this paper to conferences in order to get feedback for the findings before submitting it to a journal. Here is the abstract of our paper.
